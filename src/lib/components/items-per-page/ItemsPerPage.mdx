# ItemsPerPage

An "Items per Page" component is a user interface element often used in pagination controls within web applications. It allows users to specify the number of items or entries they want to display on a single page. This component simplifies navigation and customization for users by providing control over the amount of content shown per page. It's particularly useful for lists, tables, or search results, enhancing the user's experience by tailoring content visibility to their preferences.

### Import

```ts
import { ItemsPerPage } from 'planejape-ui-svelte';
```

### Usage

<Canvas of={ItemsPerPageStories.Default} />

Import `ItemsPerPage` component and bind the value propert, ex:

1. Define your let that will receive the current value of items per page

```html
<script>
	let ItemsPerPage = undefined;
</script>
```

2. Use the `bind` property to get or set the value of `ItemsPerPage`

```jsx
<ItemsPerPage bind:value={ItemsPerPage} />
```

**OBS**: You don't need to pass all props of `ItemsPerPage`. The value of `itemsPerPage` and `optionsQuantity` are defined by default, however, you can define it if you prefer
