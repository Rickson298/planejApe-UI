# Flex

A Flex component is a versatile layout element used in web and software development to create flexible and responsive designs. It allows content to adapt to different screen sizes and orientations. Flex components are valuable for building responsive user interfaces, enabling the easy arrangement of items within containers, and adjusting their alignment and distribution to fit various display dimensions. They simplify the design process and enhance the user experience.

### Import

```ts
import { Flex } from 'planejape-ui-svelte';
```

### Usage

<Canvas of={FlexStories.Default} />

Import `Flex` using step above and pass, and pass the flex property that you want:

```jsx
<Flex height="300px" width="50vw" padding="1rem" class="flex__container" {...$$restProps}>
	<div class="flex__child 1">Item 1</div>
	<div class="flex__child 2">Item 2</div>
	<div class="flex__child 3">Item 3</div>
</Flex>
```
